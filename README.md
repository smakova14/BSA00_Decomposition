# BSA00_Decomposition

Проект по декомпозиции бизнес-систем и анализу требований для двух кейсов: онлайн-запись в барбершоп (BRB) и сервис доставки (DLV).

## 🛠 Навыки
- Requirements Analysis
- Change Management
- Analytical thinking
- Functional & Object-Oriented Design
- Event-driven Analysis

## 📋 Описание проекта
В ходе работы я провела декомпозицию предметной области и системный анализ для двух сервисов: барбершоп (онлайн-запись) и доставка (курьерская служба).

### Что именно было сделано:
1. **Анализ предметной области**: Изучила и систематизировала источники информации (нормативные акты, статьи, интервью с менеджером барбершопа и курьером).
2. **Составление глоссария**: Определила ключевые термины и роли (мастер, клиент, курьер, диспетчер, статус заказа).
3. **Декомпозиция по видам**: Выполнила функциональную, объектную, структурную и событийную декомпозицию (демографическая пирамида, завтрак, действия курьера, активности дня).
4. **Анализ ошибок**: Нашла и исправила ошибки в декомпозиции (нарушение логики, пропуски, дублирования).
5. **Оценка изменений**: Определила влияние изменений на систему (Change Management) и приоритеты требований.
